Quick Start Guide

Get up and running with Noobly JS in minutes

1. Installation

Install Noobly JS using npm:

npm install noobly-core

2. Basic Setup

Create your main application file:

const noobly = require('noobly-core'); const app = noobly.create();

3. Start Building

Add services and start your server:

app.addService('cache'); app.listen(3000);
